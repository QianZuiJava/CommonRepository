## code-to-name 编码转名称
######  服务中国际化使用：存储编码，展示名称功能；在数据存储时，可以是编码名称 和 业务数据隔离存储

## log-anonymous 日志脱敏
######  在服务器日志打印时，避免姓名、电话、密码等敏感信息的打印；通过此工具实现匿名类别可配置
